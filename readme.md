# Minimal Store

Учебный проект интернет-магазина, выполненный на чистом HTML, CSS и JavaScript без использования фреймворков.

Пользователь может:
- просматривать каталог товаров;
- добавлять товары в корзину;
- изменять количество товаров;
- удалять товары из корзины;
- видеть итоговую сумму заказа;
- оформлять заказ через форму в модальном окне.

## О проекте

Проект реализует простой интернет-магазин в соответствии с требованиями лабораторной работы. Основной упор сделан на:
- семантическую HTML-разметку;
- адаптивную верстку;
- работу с корзиной;
- сохранение состояния корзины в `localStorage`;
- оформление заказа через форму.

## Функциональность

### Каталог товаров
- отображение списка товаров;
- у каждого товара есть название, цена и кнопка **«Добавить в корзину»**.

### Корзина
- добавление товара в корзину;
- удаление товара из корзины;
- изменение количества товара;
- автоматический пересчет итоговой суммы;
- отображение общего количества товаров;
- сохранение данных корзины в `localStorage` после перезагрузки страницы.

### Оформление заказа
- открытие формы заказа в модальном окне;
- обязательные поля:
    - имя;
    - фамилия;
    - адрес доставки;
    - контактный номер телефона;
- после успешной отправки формы выводится сообщение: **«Заказ создан!»**.

## Использованные технологии

- **HTML5**
- **CSS3**
- **JavaScript **
- **localStorage**

Фреймворки и библиотеки не использовались.
